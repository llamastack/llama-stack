# Inference

## Overview

Llama Stack supports 23 inference providers. Develop locally with Ollama, deploy to production with vLLM, or connect to a managed service. The API stays the same.

Three kinds of models are supported:
- **LLM models**: generate chat and text completions
- **Embedding models**: generate vector embeddings for semantic search
- **Rerank models**: reorder documents by relevance to a query

This section contains documentation for all available providers for the **inference** API.
