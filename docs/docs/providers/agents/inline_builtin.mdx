# inline::builtin

## Description

Implements the Anthropic Managed Agents API (v1alpha) for creating and managing agent configurations with model, tools, and system prompt specifications. This provider stores agent configurations in-memory and provides CRUD operations compatible with the Claude Agent SDK.
